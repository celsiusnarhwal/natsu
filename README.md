# natsu

natsu allows you to `sum()` Python objects of any class that implements `__add__()`.

## Installation

```bash
pip install natsu
```

## Usage

```python
from natsu import sum

sum(some_iterable)
```

`natsu.sum()` is fully backwards-compatible with the built-in `sum()` function.
